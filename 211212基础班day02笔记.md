# 8. 多情况判断分支语句(if-elif-else语句)

- ## 分支语句中只会执行第一个满足条件的分支的代码

- ## 如果上一个条件不成立，则下一个条件需要考虑上一个条件的反向条件

```python
try:
    score = int(input("请输入您的得分："))
    if 100 >= score >= 90:
        print("优秀")
    elif score >= 80:
        print("良好")
    elif score >= 70:
        print("中等")
    elif score >= 60:
        print("合格")
    elif score >= 0:
        print("不及格")
    else:
        print("你输入的分数有误")
except ValueError as e:
    print("您输入的内容有误，请输入一个数字！" + str(e))
```

- ## 怎样判断情况已经分析完全：用==**数轴**==
